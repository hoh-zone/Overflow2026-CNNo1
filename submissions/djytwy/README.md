## Description / 项目简介

Sui Lending Dashboard 是一款工具类的产品可以帮助用户一眼看到sui链上各个借贷协议的情况，让用户在sui区块链上面找寻最适合自己的借贷协议存放稳定币,
从而赚取稳定的收益。内置一键claim功能，可以让用户一键claim所有集成的借贷协议的rewards。
