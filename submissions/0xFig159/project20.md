## Description / 项目简介

DeepBook Predict Scenario Simulator provides scenario simulation for prediction markets on DeepBook. It allows users to model market outcomes, test strategies, and analyze risk before committing funds.

DeepBook Predict Scenario Simulator 为 DeepBook 上的预测市场提供场景模拟。用户在投入资金前可建模市场结果、测试策略和分析风险。
