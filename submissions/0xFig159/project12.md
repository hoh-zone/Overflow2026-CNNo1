## Description / 项目简介

Sui Package Reputation Registry provides reputation and trust scoring for Sui Move packages. It aggregates on-chain usage data, audit history, and community reviews to help developers assess package reliability and security.

Sui Package Reputation Registry 为 Sui Move 包提供信誉与信任评分。聚合链上使用数据、审计历史和社区评价，帮助开发者评估包的可靠性和安全性。
