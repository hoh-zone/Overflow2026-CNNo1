## Description / 项目简介

Walrus Dataset License Kit provides dataset licensing and management on Walrus/Sui. It enables data creators to define license terms, manage access rights, and track dataset usage with immutable Walrus-anchored records.

Walrus Dataset License Kit 在 Walrus/Sui 上提供数据集许可与管理。数据创建者可定义许可条款、管理访问权限，并通过 Walrus 锚定的不可篡改记录跟踪数据集使用情况。
