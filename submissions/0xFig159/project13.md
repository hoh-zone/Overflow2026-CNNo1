## Description / 项目简介

DeepBook Execution Quality Report provides execution quality analytics and reporting for DeepBook on Sui. It measures slippage, fill rates, price improvement, and latency to help traders evaluate and optimize their execution strategies.

DeepBook Execution Quality Report 为 Sui DeepBook 提供执行质量分析与报告。衡量滑点、成交率、价格改善和延迟，帮助交易者评估和优化执行策略。
