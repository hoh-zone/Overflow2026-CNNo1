## Description / 项目简介

Agent Permission Wallet is a permission management wallet for AI agents on Sui. It allows users to delegate granular permissions to AI agents with spending limits, time-bound access, and full audit trails stored on Walrus.

Agent Permission Wallet 是基于 Sui 的 AI Agent 权限管理钱包。用户可向 AI Agent 授权细粒度权限，包括消费限额、时间受限访问和 Walrus 存储的完整审计记录。
