## Description / 项目简介

Walrus Memory Inspector provides memory inspection and debugging tools for Walrus on Sui. It allows developers to inspect blob storage, analyze storage patterns, and debug storage-related issues.

Walrus Memory Inspector 为 Walrus on Sui 提供存储检查与调试工具。开发者可检查 Blob 存储、分析存储模式和调试存储相关问题。
