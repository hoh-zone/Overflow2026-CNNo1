## Description / 项目简介

ProofRoom provides verifiable proof and evidence management on Walrus/Sui. It enables timestamped, immutable evidence anchoring with cryptographic verification for use cases like IP protection, compliance, and dispute resolution.

ProofRoom 在 Walrus/Sui 上提供可验证的证明与证据管理。支持带时间戳的不可篡改证据锚定和加密验证，适用于知识产权保护、合规和争议解决。
