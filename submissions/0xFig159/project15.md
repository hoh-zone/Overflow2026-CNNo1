## Description / 项目简介

Keeper Receipt Network is a decentralized receipt and proof network on DeepBook/Walrus. It creates immutable, timestamped receipts for every transaction, enabling transparent audit trails for DeFi operations.

Keeper Receipt Network 是基于 DeepBook/Walrus 的去中心化收据与证明网络。为每笔交易创建不可篡改、带时间戳的收据，实现 DeFi 操作的透明审计。
