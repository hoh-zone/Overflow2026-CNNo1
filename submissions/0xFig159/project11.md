## Description / 项目简介

Move Package Diff Viewer provides a visual diff viewer for Sui Move package versions. It tracks package evolution across upgrades, highlights bytecode changes, and helps auditors and developers review smart contract modifications.

Move Package Diff Viewer 提供 Sui Move 包版本的可视化对比。跟踪包在升级过程中的演变，高亮字节码变更，帮助审计者和开发者审查合约修改。
