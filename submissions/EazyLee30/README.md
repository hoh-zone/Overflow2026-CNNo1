## Description / 项目简介

Mneme is a verifiable memory layer for AI agents on Sui. It stores agent session memories, task artifacts, and audit logs on Walrus, anchors ownership and access metadata with Sui objects, and uses Seal-style encryption policies so users can export, share, revoke, and replay agent context across tools. The MVP includes a TypeScript SDK, a memory-capsule dashboard, and a demo where two agents collaborate on one portable research memory store.
