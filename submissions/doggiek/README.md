## Description / 项目简介

### 中文

Mandate 是一个基于 Sui 的可编程权限层（Programmable Permission Layer）。

当前 AI Agent 在链上执行时普遍面临“每次操作都需要用户重新签名”的问题，难以实现真正的自主执行。

Mandate 允许用户一次授权预算、协议范围和有效期等规则，并通过链上的 Move Policy Object 强制执行这些限制。Agent 可以在授权范围内自主完成策略执行，无需为每笔交易再次请求签名。

所有执行、拦截和撤销操作都会记录在链上，用户始终保留资产所有权，并可随时撤销授权。

Mandate 提供了安全、可验证、可撤销的权限层，让 AI Agent 能够真正实现：

**一次授权，多次执行。**

### English

Mandate is a programmable permission layer for autonomous agent wallets on Sui.

AI agents are often blocked by the approval wall, where every on-chain action requires a new user signature. This makes autonomous execution difficult in practice.

Mandate allows owners to define budgets, protocol scope, expiration, and revocation rules once, then delegates limited execution authority to agents through a Move-enforced policy object.

Agents can execute autonomously within those limits, while every execution, block, and revocation remains verifiable on-chain. Ownership always stays with the user.

Mandate enables a safer model for autonomous execution:

**Sign Once, Execute Many Times.**
