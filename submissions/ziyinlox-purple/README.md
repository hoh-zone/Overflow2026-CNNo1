## Description / 项目简介

Plum Protocol is a wallet-first incident exposure and recovery coordination layer for Sui DeFi. Users can check whether a Sui wallet may be affected by verified incident scopes, review evidence-backed incident rooms, follow protocol recovery actions, and participate in recovery expectation markets tied to real incident evidence. Plum also includes a prototype Wallet Exposure SDK so wallets and dapps can integrate post-incident exposure checks without building their own incident intelligence layer.
