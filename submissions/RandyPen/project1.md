## Description / 项目简介

LockStream 让受监管的证券在链上同时做到「合规」与「流动」。证券以闭环 Token<T> 形式存在，仅能在通过 KYC 的合格持有者之间转移，是受监管的权属对象；在进入交易场所的边界上，一个合规网关适配器在链上完成 KYC、司法辖区与冻结检查后，把 Token 转换为可自由交易的 Coin<T>，进入 DeepBook 与 Cetus 的共享流动性，退出时再在同样的检查下重新封装回受监管的 Token。核心是一层一次构建、处处复用的 flows 合规信封（LegProof / Inflow<T> / Outflow<T>），让 DeepBook CLOB 与 Cetus AMM 共用同一条经审计的合规路径——合规由合约本身强制，而非可被绕过的后端。已覆盖链上 KYC（仅记录司法辖区与投资者类别，绝不上链 PII）、Reg-S 逐腿强制、发行/IPO/分红全生命周期、链上股东名册，以及 freeze / pause / reissue 的 guardian 控制。
