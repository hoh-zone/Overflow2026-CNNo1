## Description / 项目简介

PaperProof 是一个运行在 Sui mainnet + Walrus 上的可验证知识工件协议，用于把论文、数据集、技术报告、软件发布、博客和论坛内容等长期数字作品发布为稳定、可追溯、可版本化的协议工件。每个工件在 Sui 上拥有稳定链上身份和版本历史，大文件内容存储在 Walrus，并通过哈希、Blob 引用、类型化元数据和事件与链上状态绑定。项目已经提供官网、主网合约、索引器、三套 SDK、PaperProof Skill、官方 docs/blog/forum、白皮书/黄皮书/学术论文和演示视频，面向人类用户、开发者和 AI Agent 提供可验证发布与版本更新能力。
