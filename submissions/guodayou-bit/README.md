## Description / 项目简介

Arena 是一个基于 Sui 的人群共识预测市场，围绕某个命题收集目标用户回答，并让参与者在结果揭晓前围绕最终共识进行预测与验证。它把品牌调研、社区治理、内容验证和 AI 评测等高参与度场景变成可验证、可结算的链上市场。
