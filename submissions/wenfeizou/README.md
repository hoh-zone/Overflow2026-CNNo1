## Description / 项目简介

PredictGuard is the risk and hedge lifecycle workflow for DeepBook Predict PLPs,
LPs, and vault builders. It helps liquidity providers understand tail risk,
choose a hedge, execute a wallet-signed DeepBook Predict mint on Sui testnet,
read the manager state back from chain, and explain whether the hedge lifecycle
improved risk.

The final demo shows a complete workflow: PLP risk overview, exposure heatmaps,
scenario simulation, hedge recommendation, wallet readiness, a real Slush wallet
testnet mint, Sui gRPC manager readback, MarketKey decoding, lifecycle
classification, settlement accounting, and an exportable risk report.

PredictGuard 是面向 DeepBook Predict 的 PLP、LP 和 vault builder 的风险管理与
对冲生命周期工作流。它帮助资金提供者理解尾部风险、选择对冲、通过钱包签名执行
DeepBook Predict mint、从链上读回 manager 状态，并解释这个对冲生命周期是否改善
了风险。最终演示覆盖 PLP 风险总览、风险暴露热力图、压力场景、对冲建议、钱包准
备状态、Slush 钱包真实 testnet mint、Sui gRPC manager readback、MarketKey 解码、
生命周期分类、结算核算和可导出的风险报告。
