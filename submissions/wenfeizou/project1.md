## Description / 项目简介

Walrus Memory Inspector is a developer tool for making AI agent memory visible,
debuggable, and auditable. It turns agent memory into Walrus-style persistent
artifacts: raw memories, summaries, agent run traces, audit logs, timeline
entries, graph edges, and portable evidence bundles.

The demo shows how a developer can inspect conflicting memories, run an agent
with cited memory, review retrieval scores and matched tokens, resolve stale
memory by latest evidence, store the correction as an audit artifact, and export
or import the full evidence bundle for cross-session handoff.

Walrus Memory Inspector 是一个面向 AI agent memory 的开发者工具。它把隐藏在
prompt/context 里的记忆变成可持久化、可检查、可调试、可审计的 Walrus-style
artifacts，包括 raw memories、summaries、agent traces、audit logs、timeline、
graph edges 和 evidence bundle。演示流程覆盖冲突记忆检测、带引用的 agent 回答、
检索分数与 token 命中、按最新记忆修正 stale memory、审计记录，以及证据包导入导出。
