## Description / 项目简介

TxTrace is the DevTools panel Sui developers wish was built into the browser. Paste any testnet digest, and TxTrace shows every PTB command — function called, every argument, every gas cost, every Move event — with a one-click failing-step pinpoint and a plain-language root-cause from a reasoning engine. No wallet, no SDK install, no Move-source upload required. Reviewer flow: open `/`, click **Try it instantly**, watch the LLM agent decode the digest against on-chain Move source and highlight the first failing step in red with an AI-generated explanation.

TxTrace 是 Sui 开发者一直想要的内置 DevTools 面板。粘贴任意 Testnet digest，TxTrace 列出每一条 PTB 命令——调用的函数、参数、gas 成本、Move event——一键定位失败步骤，并由推理引擎给出大白话归因。无需钱包、无需 SDK、无需上传 Move 源码。评审流程：打开 `/`，点击 **Try it instantly**，看 LLM agent 解码 digest、对照链上 Move 源码、把第一处失败步骤标红并解释原因。
