## Description / 项目简介

SmolAgent makes on-chain chores hireable. Each agent is a Sui shared object you mint for 1 SUI: Airdrop Hunter sweeps your wallet and claims every open airdrop in a single PTB; Tax Bookkeeper produces a quarter-ready CSV in under a minute; Wallet Guardian watches a family wallet and pings you on risky approvals. Every agent runs as a reasoning-engine loop that signs real Sui transactions and pins a verifiable digest per run — no off-chain custody surface. Reviewer flow: open `/app`, click **Hire**, sign one PTB, watch the mascot wake up and claim three airdrops on Sui Testnet.

SmolAgent 把链上杂活变成"按需雇佣"。每个 agent 都是 1 SUI 铸造的 Sui 共享对象——空投猎手在一个 PTB 中清扫所有未领空投；记账员一分钟内生成季度可申报的 CSV；钱包卫兵守护家人钱包并在高危授权时及时提醒。Agent 都是基于推理引擎的循环，每次运行都签发真实 Sui 交易并把 digest 上链留痕，全程无托管。评审流程：打开 `/app`，点击 **Hire**，签一笔 Move 调用交易，看 mascot 苏醒并在 Sui Testnet 上领走三笔空投。
