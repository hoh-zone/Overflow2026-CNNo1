## Description / 项目简介

A Verifiable Collaborative Memory Layer for Multi-Agents, Built on Walrus
Nexus Memory is a multi-agent collaborative memory and orchestration platform built on top of the Sui blockchain and the Walrus decentralized storage protocol. 
It solves the "goldfish dilemma" of modern AI agents — memory that is ephemeral, isolated, and non-portable.
