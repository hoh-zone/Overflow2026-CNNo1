## Description / 项目简介

MemoSui 是一个 AI 驱动的链上记忆助手。用户的输入会被 AI 转化为可验证的记忆并存储在 Sui 链的 Walrus 系统中，下次访问时 AI 能读取并验证历史记忆，实现真正"长期可信的 AI 记忆"。

**痛点：**
- 当前 AI 交互无法持久记忆，记忆不可验证，容易丢失或篡改
- 用户希望 AI 能记住偏好、决策历史，提供个性化建议
- 链上存储通常复杂且昂贵，缺少轻量化方案

**核心功能：**
- **Sui 钱包登录** — 确保身份和安全
- **AI 记忆生成** — 用户输入被 AI 总结成结构化记忆（原文、摘要、hash、timestamp）
- **可验证存储** — 记忆存入 Walrus，生成链上 proof，保证不可篡改
- **Memory Timeline** — 可视化展示历史记忆
- **快速检索** — AI 可读取链上历史记忆并验证完整性

**加分功能：**
- Memory NFT — 将某条记忆 mint 成 NFT
- Timeline 动画可视化
- AI 自动提醒/推荐
