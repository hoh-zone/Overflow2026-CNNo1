## Description / 项目简介

DeepFlow 是面向 Sui 生态的 DeFi 资金流转中间件。它将生息资产提取、DeepBook 交易、结算与重新存入封装为一次 PTB 原子化执行流程，提升资金利用效率并降低多步链上交互摩擦；同时作为 AI Agent / Bot 的受控执行网关，通过滑点熔断、终点锁死和周期业务配额约束自动化资金调度。
