## Description / 项目简介

RiskPilot is a bounded Sui DeFi agent for wallet risk response. It observes a Sui mainnet wallet, scores portfolio risk, plans mitigation through a multi-agent incident room, verifies the action against explicit policy boundaries, prepares a DeepBook action proof path, and archives the signed decision trail to Walrus with optional StrategyReceipt evidence. The goal is to show how AI agents can act on Sui while wallet signatures, policy checks, Sui objects, Walrus memory, and receipts keep the agent's authority visible and verifiable.
